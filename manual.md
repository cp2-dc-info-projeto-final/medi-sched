# Manual do Usuário

Esse é o manual de usuário do site da empresa médica agenda+saúde, ele foi feito para auxiliar e facilitar a interação do cliente com o sitema. Nosso site possui login para clientes, funcionários e administrador, logout, sistema de cadastramento de clientes, funcionários e administrador, sistema de agendamento com horários e datas, edição e exlusão dos agendamnentos, página de serviços e rodapé com informações para contato.
